What is the difference between var and let?
var has a global scope while let has a block scope.
you can not redeclare let.

What is the difference between var and const?
var has a global scope while const has a block scope
you can not redclare const
you can not reassign const but can manipulate the insides of an array or object

What is the difference between let and const?
you can resassign let but not const
